# Asset license bundle

License texts and small bundled assets shipped with this node suite.

## Third-party license texts

- `FLAME_LICENSE.txt` — FLAME 2020 topology, MPI non-commercial research license.
- `SMIRK_MIT_LICENSE.txt` — SMIRK source (MIT). SMIRK's output still inherits the FLAME license because it targets FLAME topology.
- `FREEUV_LICENSE.txt` — FreeUV source + weights (CC BY-NC-SA 4.0).

## ShareAlike (FreeUV) notice

FreeUV is **CC BY-NC-SA 4.0**, which adds a *ShareAlike* clause on top of the
non-commercial constraint the rest of the pipeline already carries. Any UV
albedo generated by `FreeUVGenerate` — and any downstream asset, render,
dataset, or model trained on one — inherits CC BY-NC-SA 4.0: if redistributed,
it must be distributed under the same license. Confirmed in node tooltips via
the `i_understand_non_commercial` gate and surfaced here so users reading the
repo cold are not surprised by the viral clause.

## Bundled data

- `freeuv_reference_uv.jpg` — 512×512 neutral UV reference used as the default
  `reference_uv` input when `FreeUVGenerate` is run without an explicit
  reference. Not checked in yet; drop the asset into this directory before
